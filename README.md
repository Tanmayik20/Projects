# Amazon Homepage Clone

A responsive Amazon Homepage Clone built using HTML and CSS. This project replicates the basic design and layout of Amazon's homepage, including the navigation bar, search section, product categories, and footer.

## Features

- Responsive Design
- Navigation Bar
- Search Bar
- Product Sections
- Footer Section
- User-Friendly Interface

## Technologies Used

- HTML5
- CSS3

## Project Purpose

This project was developed to improve front-end web development skills and gain hands-on experience in creating modern e-commerce website layouts.
